Utilities

MDF includes several pre-installed utilities that make your life easier. You can start most of them directly from the top bar of the GUI, or you go to MasterworkDwarfFortress/utilities.

::: Dwarf Therapist :::

Dwarf Therapist is one of the best tools for DF. You can easily select and deselect labors, see skills and attributes and sort your dwarves. It gives you a much needed improvement of the GUI and makes handling immigration waves trivial.

::: DFhack & DFusion:::

DF Hack is a program that directly interacts with the running DF-Game. Most people use it for cheating, but it can be a great utility to easy the management of greater fortresses, fix glitches, help your FPS and is an invaluable tool for modders, helping them to test their changes. It starts up automatically. If you want to disable DF Hack, simply find and delete SDL.dll and rename SDLreal.dll in your DF directory to SDL.dll, to replace the old one.

See the included readme for more info.

IMPORTANT: The poison coating and steam engine both do require dfhack to be active.

This mod also adds 7 custom plugins written by falconne. These are only GUI improvements and do not change the game balance. For more infos see here or the included readme.

Another set of plugins comes from Vjek, they are mostly to alter specific dwarves. You can find the full readme on these here.

::: Stonesense :::

Stonesense is an isometric visualizer for DF, using HD-textures. Very good for Screenshots and succession games. It is included in the dfhack pack.

See the included readme for more info.

::: Soundsense :::

Soundsense runs in the background of DF and reads out the gamelog. It then play fitting sounds, music and voices. It essentially gives DF an entire sound system. The included version is only the software, you have to download and install the soundpack you want yourself. It should be around 200mb. The links can be found on the soundsense website, or by running the program.

See the included readme for more info.

You can download the latest soundpack here (200mb)

::: Legends Viewer :::

You can view Legends Mode even in a running game with this handy tool. A GUI helps finding and exploring the legends of the world you play in. It includes an internal search engine to help you follow different characters.

See the included readme for more info.

::: Perfect World :::

Perfect World allows to create pre-designed world-gens for DF. All world-gens included in the mod have been made with this tool. You can now control savagery, volcanoes and biomes almost exactly.

See the included readme for more info.

::: Isoworld :::

Isoworld is a small program that shows you world maps in an isometric view. You have to export the detailed maps from Legends mode.

See the included readme for more info.

::: Quickfort :::

Quickfort assigns designations and furniture from templates or blueprints that you can load into it. This helps building extensive architecture and allows to establish a fortress quickly. This is a great help to establish a fortress and safe time with designations.

See the included readme for more info.

::: Creature Generator:::

The random creature script allows the creation of 10 to 1000 new creatures. These are procedually generated, similar to titans, demons and forgotten beasts. This allows for random surface and caverns creatures, and even civilizations. They can form entities, if they are smart enough. This utility has its own section in the GUI under creatures. You can choose the amount of creatures, and if you only want only creatures or entities as well.

See the included readme for more info.

::: Large Address Aware Patcher :::

This little .exe allows DF to use up to 4gb of virtual memory. This is mostly used for worldgen. You just have to locate the DwarfFortress.exe and run the patcher. Please use this only if you have more then 2gb virtual memory/RAM.

::: DF Story Maker :::

A small .exe that helps succession forts. You need to direct it to the Dwarf Fortress folder, and it will find the gamelog.txt. You can set search filters for different topics and a new file will be created that only lists gamelog entries for the specific topic.

::: Announcement Filter :::

Another small utility by theGazelle. It opens an external window that reads out DF announcements, which you can see separate from DF. Good for running DF on background, or for multiple monitor setups. Allows to filter out unwanted entries, for example job cancellations.

::: DFterm 3 - Multiplay :::

Currently in an early beta stage, this program allow several players to view and control the same fort at the same time. Please see the forum thread for more info. The GUI has a link on the top right. Once this program is more stable and refined, I will include it directly in the download package.
